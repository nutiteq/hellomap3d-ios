Getting Started

1. Create new project with Nutiteq SDK framework

Summary: Create new project, add SDK framework (binary and headers) and assets to it, and make sure that builder will find new framework .h files. Also add dependencies: GLKit and libz.

  1. Create a new Xcode project. 'File' -> 'New' -> 'Project' -> 'Single View application'.
  1. Copy Nuti.framework to the project root directory.
  1. Add dependencies: GLkit.framework and libz.dynlib frameworks (Build Phases > Link Binary with Libraries)
